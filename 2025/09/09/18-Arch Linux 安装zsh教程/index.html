Arch Linux 安装 zsh（终端美化）

Created2025-09-09|Updated2025-09-09|Linux

1- 杨CC对工具的评价

Linux下，zsh要比nushell好用很多，所以我决定在arch中，放弃nushell,转为zsh,从而书写了这一篇文章。

2- 安装前的系统参数介绍

这是我的电脑信息，我需要在这套配置中，进行安装arch.

3- 开始安装

3.1 安装基础文件

  • sudo pacman -Syu
  • 更新系统包 或者使用yay 也可以
  • sudo pacman -S zsh
  • 安装zsh
  • 安装完成zsh 后 如下图
  • zsh –version
  • 验证是否成功安装
  • 出现了上方图片中的效果，则为安装成功。
  • 将当前用户的默认shell 更改为zsh
  • chsh -s $(which zsh)
  • 安装oh-my-zsh
  • sh -c “$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)”
  • 配置zsh
  • vim ～/.zshrc
  • 注意：在此配置文件中，是可以设置主题等相关文件，如下：
  • 更改主题：ZSH_THEME=”agnoster”（需要安装 Powerline 字体）
  • 添加插件：plugins=(git docker archlinux)
  • 设置别名：alias ll=’ls -la’
  • 安装powerline字体
  • sudo pacman -S powerline-fonts # 防止出现乱码
  • 安装插件并启用
  • git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
  • git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
  • 然后在.zshrc中启用这个插件
  • plugins=(git zsh-autosuggestions zsh-syntax-highlighting)
  • 应用配置更改。
  • 配置完成后的截图
  • 我们发现，这个页面不是很漂亮，所以我们需要更换主题。

3.2 一些还不错的zsh主题

  • powerlevel10k（极客专属，高度自定义）

    炫酷，但是有点花里胡哨，需要Nerd字体。

    安装： git clone –depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

    修改： .zshrc

    ZSH_THEME=”powerlevel10k/powerlevel10k”

    截图：


    如果出现了符号异常，请运行：sudo pacman -S nerd-fonts-meslo
  • bira 极简风格的主题
  • 直接修改.zshcr文件
  • ZSH_THEME=”bira”
  • 重新载入
  • source ~/.zshrc
  • 截图：

  • bullet-train 一个集合了多种信息的zsh主题
  • 特点：类似 Powerline 的分段式设计，可显示 Git 分支、Python 虚拟环境、Node.js 版本等，支持自定义颜色和显示顺序。
  • 安装：git clone https://github.com/baskan/bullet-train-oh-my-zsh-theme.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/bullet-train
  • 修改 ~/.zshrc：
  • ZSH_THEME=”bullet-train”
  • 配置示例：
  • bullet_train_prompt_order=(dir git ruby python) # 调整显示顺序
  • bullet_train_git_colorize_dirty=true # 自动变色

  • 最后，我还是选择了下图的终端：
  • 该终端为：nushell
  • 其他主题我找了一圈，没有特别好看的，如果需要主题，请访问：https://github.com/ohmyzsh/ohmyzsh/wiki/Themes 自行寻找，并安装即可。

4- 结束语

到这里，其实已经安装完成了，插件的话，上方已经添加了高亮和补全的插件，如果需要其他插件，请自行到zsh官网进行下载，在此，我不再过多赘述。
