    WinBoa - 最强Linux虚拟机，没有之一。

    这款工具可以让你无缝运行Windows应用，当然也包括游戏。

2- 工具简单介绍

  • 支持所有的Linux环境，包括但不限于：debian\Fedora\arch等多种Linux发行版。

  • 您可以在ycc77.cn中的-其他工具-电脑工具中找到WinBoat，并使用它。

  • 当然，如果您是arch的发行版，您就不需要去寻找其他连接了。

  • 这一篇文章可以帮您完成该工具的构造。

  • 好了，我们来看一下截图。

  • 或者我们i来看夸克网盘。

  • 很好用，非常不错。下面我们来正式开始安装。

3- 特性

    说明：WinBoat 目前处于测试阶段，因此可能会偶尔遇到问题和错误。如果您决定尝试，应该能够适应一定程度的故障排除，但我们仍然鼓励您尝试一下。

    特性：    

  • 🎨 优雅的界面：时尚直观的界面，将 Windows 无缝集成到您的 Linux 桌面环境中，让您感觉像原生体验
  • 📦 自动安装：通过我们的界面进行简单的安装过程 - 选择您的偏好和规格，让我们处理其余部分
  • 🚀 运行任何应用：只要能在 Windows 上运行，就能在 WinBoat 上运行。在 Linux 环境中，像原生操作系统级窗口一样畅享各种 Windows 应用。
  • 🖥️ 完整的 Windows 桌面：在需要时访问完整的 Windows 桌面体验，或运行无缝集成到 Linux 工作流程中的单个应用程序
  • 📁 文件系统集成：您的主目录安装在 Windows 中，从而允许在两个系统之间轻松共享文件，没有任何麻烦
  • ✨ 还有更多：智能卡直通、资源监控以及定期添加的更多功能

4- 运行的条件

在运行 WinBoat 之前，请确保您的系统满足以下要求：

  • 内存：至少 4 GB 内存

  • CPU：至少 2 个 CPU 线程

  • 存储空间：至少 32 GB 可用空间/var

  • 虚拟化：在 BIOS/UEFI 中启用 KVM

    • 如何启用虚拟化

  • Docker：容器化所需

    • 安装指南

  • Docker Compose v2：与 docker-compose.yml 文件兼容所需

    • 安装指南

  • Docker 用户组：将您的用户添加到docker组

    • 设置说明

  • FreeRDP：远程桌面连接所需（请确保您拥有包含声音支持的 3.xx 版本）

    • 安装指南

  • 内核模块：必须加载 iptables模块iptable_nat

    • 模块加载说明

    是的，在你运行WinBoat之前，必须满足上方的条件，如果没有满足上方条件，请点击链接，进行修复这些条件。

5- 下载方式

5.0 常用下载（推荐）

  • 如果你是 debian发行版或者fedora发行版，你可以访问：Releases · TibixDev/winboat · GitHub

  • 进入页面后，如果你是debian发行版请选择.deb，如果你是fedora发行版，请选择.rpm文件。

  • 如果.deb 和.rpm无法使用，请下载applmage文件，并运行。

  • arch发行版，那么请按照下方操作。

  • 国内用户请提前准备VPN，国外用户不需要准备VPN。

  • # 先安装 yay命令。
sudo pacman -Syu dase-devel git 

# 克隆yay仓库
git clone https://aur.archlinux.org/yay.git

# 进入目录并构建安装yay
cd yay
makepkg -si

# 验证是否安装成功
yay --version
# 当出现 yay相关说明后，就代表安装成功了。

  • 如果上方三种方法依然无法使用，m那么请下载.applmage文件。

  • 下载.applmage文件后，使用：chmod +x winboat_(版本号和其他说明).applmage

5.1 本地构建WinBoat（不建议）

  • 为了构建，你需要在系统上安装 NodeJS 和 Go
  • 克隆 repo（git clone https://github.com/TibixDev/WinBoat）
  • 安装依赖项（npm i）
  • 使用以下方式构建应用程序和客户服务器npm run build:linux-gs
  • dist您现在可以在 AppImage 和未打包的变体下找到构建的应用程序

5.2 源码部署（二开可以选择这个）

  • 确保满足先决条件
  • 此外，为了进行开发，你需要在系统上安装 NodeJS 和 Go
  • 克隆 repo（git clone https://github.com/TibixDev/WinBoat）
  • 安装依赖项（npm i）
  • 构建访客服务器（npm run build-guest-server）
  • 运行应用程序（npm run dev）
