Windows-Linux-amc全系统安装vmware虚拟机

Created2025-09-14|Updated2025-09-14|虚拟机

1- 杨CC有话说

vmware虚拟机在17版本以后,就已经全面开放,不再需要购买,这篇文章,将介绍如何在Windows、Linux、amc系统安装vmware虚拟机.

2- 安装vmware虚拟机

2.1 Windows安装vmware虚拟机

  • 准备工作

    确认系统要求

    操作系统：Windows 10/11 专业版 / 企业版（家庭版需开启 Hyper-V 兼容模式，部分功能受限）。

    硬件：CPU 支持虚拟化技术（Intel VT-x 或 AMD-V），并在 BIOS 中启用（重启电脑按 F2/F10/Del 进入 BIOS，找到 “Virtualization Technology” 设为 Enabled）。

    内存：至少 4GB（推荐 8GB 以上，否则虚拟机运行卡顿）。


下载 VMware 安装包

访问 VMware 官网，选择 “Workstation Pro”，根据系统版本（64 位）下载对应安装包（如 VMware-workstation-full-17.0.0.exe）。

如果无法下载,可以访问垮克网盘下载。

  • 然后找到:VMware Workstation 文件夹,进入.就可以下载安装了.

  • 双击打开安装包,然后点击下一步.

  • 一路点击下一步,直到安装完成.

  • 安装完成以后,点击桌面上的vmware图标,打开vmware.

  • 至此,Windows安装vmware虚拟机完成.

  • 检查虚拟化是否启用

  • 打开 VMware 后，点击 “编辑”→“首选项”→“处理器”，若提示 “虚拟化引擎：Intel VT-x/AMD-V 已启用”，则硬件支持正常；若提示 “未启用”，需重新进入 BIOS 开启虚拟化技术。

  • 安装系统,我们不再进行赘述,可以自行百度.

  • 如果你找不到对应的系统镜像,可以访问垮克网盘下载.

2.2 MACOS 安装vmware虚拟机

  • 准备工作

  • 确认系统兼容性

  • Intel 芯片 Mac：支持 VMware Fusion 11 及以上版本，兼容 macOS 10.15（Catalina）及更高。

  • Apple Silicon 芯片 Mac（M1/M2/M3）：需使用 VMware Fusion 13 及以上版本（首次支持 Apple 芯片），系统需 macOS 12（Monterey）及更高。
    硬件要求：至少 4GB 内存（推荐 8GB 以上），空闲磁盘空间≥20GB（用于虚拟机系统）。

  • 下载 VMware Fusion

  • 访问 VMware 官网，选择对应版本：

  • 免费版：VMware Fusion Player（个人非商用免费，功能简化）。

  • 专业版：VMware Fusion Pro（需购买，适合开发 / 企业用户）。

  • 下载后得到 .dmg 安装包（如 VMware-Fusion-13.0.2.dmg）。

  • 安装 VMware Fusion 步骤

  • 挂载安装包

  • 双击下载的 .dmg 文件，打开后会显示安装界面，包含 VMware Fusion 应用图标。

  • 安装到应用程序

  • 将 VMware Fusion 图标拖动到右侧的 应用程序 文件夹中（类似普通 Mac 软件安装），等待复制完成（约 1-2 分钟）。

  • 首次启动与权限设置

  • 打开 应用程序 文件夹，找到并双击 VMware Fusion。

  • 首次启动会提示 “无法打开，因为它来自未知开发者”：

  • 解决方法：打开 系统设置 → 隐私与安全性 → 拉到最下方，找到 “VMware F- usion”，点击 “仍要打开”（可能需要输入 Mac 密码）。

  • 许可协议与激活

  • 勾选 “我同意许可协议”，点击 “继续”。

  • 若使用免费版（Player）：选择 “个人免费使用”，点击 “继续”（无需密钥）。

  • 若使用专业版（Pro）：输入购买的许可证密钥，点击 “激活”。

  • 完成初始化设置

  • 可能会提示 “是否允许访问网络”“是否允许访问辅助功能”，点击 “允许”（确保虚拟机网络和外设正常工作）。

  • 首次启动后，会显示主界面，提示 “创建新虚拟机”，此时安装完成。

  • 启用虚拟化支持（Apple Silicon 芯片必看）

  • Apple Silicon 芯片的 Mac 需额外启用 “系统扩展” 权限：

  • 打开 系统设置 → 隐私与安全性 → 安全性 → 找到 “已阻止系统软件来自‘VMware, Inc.’”，点击 “允许”。

  • 按提示重启 Mac，重启时按住电源键直到出现 “启动选项”，选择 “选项”→ 进入恢复模式。

  • 在恢复模式中，打开 实用工具 → 启动安全性实用工具 → 选择 “降低安全性”→ 勾选 “允许用户管理的系统软件和内核扩展”，点击 “确定”，重启 Mac。
